# Pyramid Decode

This is a python implementation of a pyramid decode. It allows the creation and printing of a pyramid decode. 

Example text file:

```
1 I
2 am
3 love
4 a
5 you
6 computers
```

Example output:

```
I love computers
```
formation of pyramid:
```
1
2 3
4 5 6
```

Decoding of pyramid: 
```
1 I
3 love
6 computers
```
